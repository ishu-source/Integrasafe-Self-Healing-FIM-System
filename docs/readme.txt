## 📌 Overview

IntegraSafe is a self-healing file integrity monitoring system designed to detect unauthorized file modifications and automatically restore original files using secure backup and hash verification techniques.

---

## 🚀 Key Features

* 🔍 Real-time file monitoring
* 🔐 SHA-256 hash-based integrity verification
* ♻️ Automatic file recovery (self-healing mechanism)
* 🚨 Unauthorized change detection & alerting
* 📁 Secure backup management
